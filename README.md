# Install project
Run lần lượt các CMD sau:
```
    1. composer install
    2. npm install
    3. npm run dev
    4. cp .env.example .env
    5. php artisan config:cache
    6. php artisan cache:cache
    7. php artisan route:cache
    8. php artisan migrate
    9. php artisan db:seed
    10. php artisan serve
```
